# 信息采集系统

**crawler-system-vue3**  
版本：v2.0.0

## 📌 项目简介

信息采集系统是一个，用于采集、查看、推送爬虫信息的集成平台。
90%的代码保留了，删除了部分涉密信息，核心代码都在，大概率可以跑起来，后续有时间再完善

## 📦 使用说明

### 1. 克隆项目
### 2. 安装依赖
```bash

npm install
```
> 推荐使用 npm 或 yarn，确保 Node.js 版本 >= 14.18.0

### 3. 启动开发服务器
```bash

npm run dev
```
启动后访问：http://localhost:5173

### 4. 构建生产版本
```bash

npm run build
```
构建后的文件将输出到 `/www/` 目录。

## 🚀 部署说明

### 静态部署（推荐）
本项目构建后为纯静态文件，可部署到任意静态服务器或 CDN：
- Nginx
- Apache
